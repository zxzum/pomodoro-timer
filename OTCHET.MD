## Архитектура VS Code Extensions

### Основные компоненты

1. **Манифест (package.json)** - описывает расширение
2. **Точка входа (extension.ts)** - функции `activate()` и `deactivate()`
3. **Extension API** - интерфейс для взаимодействия с редактором
